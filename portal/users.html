Add New User

To create a new staff account, use Supabase's invite feature.

Step-by-step:

  1. Go to supabase.com/dashboard
  2. Open your komtek-tickets project
  3. Click Authentication → Users → Invite user
  4. Enter the user's email and send the invite
  5. They set their own password via the emailed link
  6. Their profile appears here — set their role below
